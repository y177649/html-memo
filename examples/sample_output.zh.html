关于 html-memo 技能

html-memo 是一款在 Claude Code 中运行的自定义技能。只需一条命令，便能将日常的工作日志与学习笔记保存为「衬线字体 × 深色模式」风格的优雅 HTML 文件。它的目标，是把那些常以纯 Markdown 形式被遗忘的信息，升华为真正值得反复翻阅的「文档」。

笔记的巅峰并非写下的那一刻，其价值在重新阅读时方才重生。正因如此，视觉之美支撑着思考的连续性。

设计规格

背景采用比纯黑更柔和的 #111418，摒弃装饰性渐变，仅以细微噪点构成平整而现代的版面。长时间阅读也不易疲劳，而冷色调强调色 #5BB3C9（一种带来沉静与专注的青色）则在纸面上静静生辉。

排版

正文指定使用衬线字体，为文本赋予品格与易读性。章节编号与引号搭配 Cormorant Garamond，以拉丁字母的神韵为版面带来节奏。字号刻意保持克制，留下安静而雅致的留白。

元素作用处理方式
标题章节分隔自动编号
首段导入首字下沉
引用强调洞见装饰边框
代码技术性表述等宽字体区分

使用方法

在 Claude Code 的提示框中输入 /html-memo，再接着写下想要记录的内容即可。技能会自动生成 HTML 文件，并保存到 memos/ 目录。 

/html-memo 整理今天的开发回顾与明天的任务

生成的文件在以下场景中尤为出彩：

  • 希望把日常开发日志留存为日后可反复翻阅的记录时
  • 希望让学习笔记不只是随手涂写，而是作为「文档」积累时
  • 希望把调研结果与设计决策整理成可与团队共享的体例时

多语言支持

技能会自动判别你所使用的语言。用日语下达指令便生成日语笔记，用中文下达指令便生成中文笔记，二者皆采用同样精美的版式。

既是为写作者打造的工具，也为阅读者的体验而设计。这正是 html-memo 所追求的境界。
