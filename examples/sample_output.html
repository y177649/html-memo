html-memoスキルについて

html-memo は、Claude Code 上で動作するカスタムスキルです。日々の作業ログや学習メモを、コマンド一発で「明朝体 × ダークモード」のエレガントな HTMLファイル として保存できます。Markdownのまま放置されがちな情報を、読み返したくなる「資料」へと昇華させることを目的としています。

メモは書いた瞬間がピークではなく、読み返したときに価値が再生される。だからこそ、見た目の美しさが思考の継続性を支える。

デザイン仕様

背景色には純黒より柔らかい #121212 を採用し、温かみのあるラジアルグラデーションと微細なノイズテクスチャを重ねています。長時間の閲覧でも目が疲れにくく、アクセントカラーの #D97757（Claudeオレンジ）が紙面に静かに映えます。

タイポグラフィ

本文には明朝体を指定し、日本語テキストに品格と読みやすさを与えます。見出しの連番や引用符には Cormorant Garamond を組み合わせ、欧文の表情で誌面にリズムを生み出します。

要素役割処理
見出しセクション区切り連番を自動付与
段落冒頭導入ドロップキャップ化
引用インサイトの強調装飾枠で囲む
コード技術的な記述等幅フォントで区別

使い方

Claude Code のプロンプトに /html-memo と入力し、メモにしたい内容を続けて記述するだけです。スキルが自動的にHTMLファイルを生成し、memos/ ディレクトリに保存します。 

/html-memo 今日の開発の振り返りと明日のタスクをまとめて

出力されたファイルは、次のような場面で力を発揮します。

  • 日々の開発ログを、後から読み返せる記録として残したいとき
  • 学習メモを、ただの走り書きではなく「資料」として蓄積したいとき
  • 調査結果や設計判断を、チームに共有できる体裁で整えたいとき

こんな人におすすめ

プレーンテキストでは物足りないが、ドキュメントツールを開くのは大袈裟──そんな 「日常のメモと正式な資料の中間」 を求めている方に最適です。

書き手のための道具でありながら、読み手の体験までデザインする。それが html-memo の目指すところです。
