html-memoスキルについて

概要

html-memo は、Claude Code 上で動作するカスタムスキルです。日々の作業ログや学習メモを、コマンド一発で「明朝体 × ダークモード」のエレガントな HTMLファイル として保存できます。

Markdownのまま放置されがちな情報を、読み返したくなる「資料」へと昇華させることを目的としています。

メモは書いた瞬間がピークではなく、読み返したときに価値が再生される。だからこそ、見た目の美しさが思考の継続性を支える。

デザイン仕様

背景色には純黒より柔らかい #121212 を採用しています。長時間の閲覧でも目が疲れにくく、アクセントカラーの #D97757（Claudeオレンジ）が自然に映えます。

フォントは明朝体を指定し、日本語テキストに品格と読みやすさを与えます。英数字には等幅フォントを組み合わせ、コードスニペットとの視覚的な区別を明確にします。

使い方

Claude Code のプロンプトに /html-memo と入力し、メモにしたい内容を続けて記述するだけです。スキルが自動的にHTMLファイルを生成し、memos/ ディレクトリに保存します。 

/html-memo 今日の開発の振り返りと明日のタスクをまとめて

こんな人におすすめ

プレーンテキストでは物足りないが、ドキュメントツールを開くのは大袈裟──そんな 「日常のメモと正式な資料の中間」 を求めている方に最適です。

書き手のための道具でありながら、読み手の体験までデザインする。それが html-memo の目指すところです。
