關於 html-memo 技能

html-memo 是一款在 Claude Code 中運行的自訂技能。只需一道指令，便能將日常的工作日誌與學習筆記儲存為「襯線字體 × 深色模式」風格的優雅 HTML 檔案。它的目標，是把那些常以純 Markdown 形式被遺忘的資訊，昇華為真正值得反覆翻閱的「文件」。

筆記的巔峰並非寫下的那一刻，其價值在重新閱讀時方才重生。正因如此，視覺之美支撐著思考的連續性。

設計規格

背景採用比純黑更柔和的 #111418，摒棄裝飾性漸層，僅以細微雜訊構成平整而現代的版面。長時間閱讀也不易疲勞，而冷色調強調色 #5BB3C9（一種帶來沉靜與專注的青色）則在紙面上靜靜生輝。

排版

內文指定使用襯線字體，為文字賦予品格與易讀性。章節編號與引號搭配 Cormorant Garamond，以拉丁字母的神韻為版面帶來節奏。字級刻意保持克制，留下安靜而雅致的留白。

元素作用處理方式
標題章節分隔自動編號
首段導入首字放大
引用強調洞見裝飾邊框
程式碼技術性敘述等寬字體區分

使用方法

在 Claude Code 的提示框中輸入 /html-memo，再接著寫下想要記錄的內容即可。技能會自動產生 HTML 檔案，並儲存到 memos/ 目錄。 

/html-memo 整理今天的開發回顧與明天的任務

產生的檔案在以下情境中格外出色：

  • 希望把日常開發日誌留存為日後可反覆翻閱的記錄時
  • 希望讓學習筆記不只是隨手塗寫，而是作為「文件」累積時
  • 希望把調查結果與設計決策整理成可與團隊共享的體例時

多語言支援

技能會自動判別你所使用的語言。用日語下達指令便產生日語筆記，用中文下達指令便產生中文筆記，兩者皆採用同樣精美的版式。

既是為寫作者打造的工具，也為閱讀者的體驗而設計。這正是 html-memo 所追求的境界。
