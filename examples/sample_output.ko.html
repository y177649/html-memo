html-memo 스킬 소개

html-memo는 Claude Code 위에서 동작하는 커스텀 스킬입니다. 명령어 한 번으로 일상의 작업 로그와 학습 메모를 「명조체 × 다크 모드」의 우아한 HTML 파일로 저장할 수 있습니다. Markdown인 채로 방치되기 쉬운 정보를, 다시 읽고 싶어지는 「문서」로 승화시키는 것을 목표로 합니다.

메모는 작성한 순간이 정점이 아니라, 다시 읽을 때 그 가치가 되살아납니다. 그렇기에 시각적인 아름다움이 사고의 연속성을 떠받칩니다.

디자인 사양

배경에는 순수한 검정보다 부드러운 #111418을 채택하여, 장식적인 그라데이션을 배제하고 미세한 노이즈만을 입힌 플랫하고 모던한 면으로 마무리했습니다. 장시간 열람해도 눈이 편안하며, 한색 계열 강조색 #5BB3C9(진정과 집중을 돕는 시안)이 지면 위에서 조용히 빛납니다.

타이포그래피

본문에는 명조체를 지정하여 텍스트에 품격과 가독성을 부여합니다. 섹션 번호와 인용 부호에는 Cormorant Garamond를 조합하여, 라틴 문자의 표정으로 지면에 리듬을 만들어 냅니다. 글자 크기는 절제하여, 조용하고 품위 있는 여백을 남겼습니다.

사용법

Claude Code 프롬프트에 /html-memo를 입력하고, 메모하고 싶은 내용을 이어서 적기만 하면 됩니다. 스킬이 자동으로 HTML 파일을 생성하여 memos/ 디렉터리에 저장합니다. 

/html-memo 오늘의 개발 회고와 내일의 작업을 정리해줘

생성된 파일은 다음과 같은 상황에서 진가를 발휘합니다.

  • 일상의 개발 로그를, 나중에 다시 읽을 수 있는 기록으로 남기고 싶을 때
  • 학습 메모를, 단순한 끄적임이 아닌 「문서」로 축적하고 싶을 때
  • 조사 결과나 설계 판단을, 팀에 공유할 수 있는 형식으로 정리하고 싶을 때

다국어 대응

스킬은 입력된 언어를 자동으로 판별합니다. 한국어로 지시하면 한국어 메모가, 영어로 지시하면 영어 메모가 동일한 아름다운 레이아웃으로 생성됩니다.

글쓴이를 위한 도구이면서, 읽는 이의 경험까지 디자인한다. 그것이 html-memo가 지향하는 바입니다.
