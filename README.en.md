# html-memo

**Turn your daily notes and work logs into elegant HTML files styled in "serif typography × dark mode" — with a single command. Just type `/html-memo` in the Claude Code prompt, and a document worth revisiting is generated for you.**

> A Claude Code custom skill that elevates information often left to rot as plain Markdown into refined documents. No installation required — just drop it into your `commands` folder.

## Features

| | |
|---|---|
| **Dark mode base** | Uses `#111418`, softer than pure black, so your eyes stay comfortable during long reading sessions |
| **Serif typography** | Premium typography that gives text dignity and readability |
| **Cool-toned accent** | A calming, focus-friendly cyan `#5BB3C9` naturally highlights key insights |
| **No installation** | Works by dropping just one file into your `commands` folder |
| **Multilingual** | Auto-detects the language you prompt in (Japanese / English / Korean / Simplified Chinese / Traditional Chinese) and writes the memo in the same language |

## Quick Start

No need to place files manually. **Just hand Claude Code the URL of this repository and ask it to set the skill up for you.**

Launch Claude Code and give it an instruction like:

```text
Set up the skill from this repository so I can use it:
https://github.com/y177649/html-memo
```

Claude Code reads the repository, places `commands/html-memo.md` into your project's `commands` folder, and the `/html-memo` command is ready to use.

---

## Manual Installation

If you prefer to set things up by hand, without Claude Code:

1. Download `commands/html-memo.md` from this repository.
2. Place it inside the `commands` folder in your project's root directory. (Create the folder if it doesn't exist.)

---

## Usage

Launch Claude Code and run the command as follows:

```bash
> /html-memo Summarize today's development progress and the upcoming challenges
```

A beautiful HTML file will be automatically generated in the `memos/` directory.

---

## Setting Up a Comfortable Preview Environment (Recommended)

To preview HTML files inside your editor with a single click, just like Markdown, we recommend installing Microsoft's official extension.

1. When you open this project in VS Code / Cursor, a pop-up will appear in the bottom-right corner recommending the `Live Preview` extension — please install it.
2. When you open an HTML file, a preview icon (a magnifying glass, or a "Show Preview" button) appears in the top-right of the editor. Just click it to check your design directly inside the editor, without opening a browser.
